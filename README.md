# MatchYou - 基于兴趣标签的高校社团招新智能匹配平台

## 项目介绍
MatchYou 是一款 NoCode 低代码平台开发的高校社团招新与交流平台，面向新生、社团管理员、学校管理员三方角色，实现社团智能推荐、在线报名、信息管理、数据统计、交友论坛一体化功能。

## 在线体验地址（可直接访问）
https://interest-based-nexus.nocode.host

## 功能模块

### 👩‍🎓 新生端
- 用户注册、登录
- 兴趣标签选择，生成个人画像
- 智能算法推荐适合的社团
- 社团浏览、搜索、筛选
- 一键报名、查看报名状态
- 通知中心、个人中心
- 交友论坛（同学交流、发帖、评论）

### 👥 社团管理员端
- 社团信息管理（查看模式 / 编辑模式）
- 招新信息发布与管理
- 报名数据查看、状态处理
- 接收学校通知
- 数据统计展示

### 🏫 学校管理员端
- 社团信息审核
- 全局公告发布
- 平台数据总览
- 用户与权限管理

## 技术栈
- NoCode 低代码平台
- React 前端框架
- Supabase 后端云数据库
- GitHub 源码托管

## 项目亮点
- 三端分离，角色权限清晰
- 智能兴趣匹配算法
- 完整闭环：报名—审核—通知—交流
- 交友论坛增强用户互动
- 响应式界面，支持手机、电脑访问

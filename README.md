# Match You 社团招新智能匹配平台
✨ 一站式高校社团招新 & 智能匹配 & 交友交流平台

## 项目介绍
Match You 是一款专为高校打造的**社团招新智能匹配平台**，解决传统招新信息不对称、流程繁琐、匹配低效、管理混乱等问题。
平台面向**新生、社团管理员、学校管理员**三方用户，实现智能推荐、在线报名、流程管理、数据统计、交友论坛一体化服务。

## 核心功能

### 👩‍🎓 新生端
- 账号注册与身份认证
- 兴趣标签 + 兴趣测评生成个人画像
- **智能匹配算法**推荐最适合的社团
- 社团浏览、筛选、搜索、详情查看
- 一键报名、查看报名进度、面试通知、录取结果
- 通知中心、个人中心、资料编辑
- **交友论坛**：与同好新生交流、提问、分享

### 👥 社团管理员端
- 社团信息管理（查看模式 / 编辑模式）
- 招新信息发布、自定义报名表
- 报名管理、状态流转、面试安排
- 招新数据看板、数据统计与导出
- 新生咨询回复、评价管理
- 系统通知接收

### 🏫 学校管理员端
- 社团招新审核与监管
- 全校招新数据总览与分析
- 用户权限管理（禁用/恢复账号）
- 评价管理、公告发布
- 平台运行监控

## 技术栈
- 前端：React + JSX + CSS
- 后端/数据库：Supabase
- 部署：GitHub Pages
- 架构：三端分离（新生端 / 社团管理端 / 学校管理端）

## 智能匹配算法
匹配度 = 学生兴趣标签与社团标签重合数量 / 学生总标签数 × 100%
- 按匹配度从高到低排序展示
- 80% 以上标记热门 🔥
- 支持多维度筛选：类别、活动频率、面试难度、招新状态

## 主要特色
✅ 真实数据驱动，无模拟假数据  
✅ 三端权限分离，安全规范  
✅ 智能推荐，提升招新效率  
✅ 报名—面试—录取—通知全流程闭环  
✅ 交友论坛，促进新生交流  
✅ 响应式设计，支持手机 / PC 双端  
✅ 管理后台默认查看模式，更规范  

## 项目结构
- 项目首页
- student 新生端（登录、首页、报名、通知、个人中心、论坛）
- club 社团管理员端
- school 学校管理员端


## 使用说明
1. 新生：注册 → 完善兴趣 → 智能匹配 → 报名社团 → 查看进度
2. 社团：登录 → 完善信息 → 发布招新 → 管理报名 → 查看数据
3. 学校：登录 → 审核招新 → 监管数据 → 发布公告
